
A homework project for my Computer Science II class that 
reads employee info from a text file and sorts/displays
info by user choice.

Author: Hannah Seccia
Date of Creation: Feburary 2018 

Description: 

This project reads a variable number of employee's info, such
as SSN, name and building number. The project is structured
as a menu that lets the user display certain attributes of
an employee or find them by search.
